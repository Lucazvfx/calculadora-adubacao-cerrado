# Adubação Cerrado

Pacote Python para cálculo de recomendação de adubação para as culturas de milho, soja e pastagem, baseado nos manuais da Embrapa Cerrados.

## Instalação

```bash
pip install -r requirements.txt
python app.py
.\venv\Scripts\activate
git add .
git add templates/index.html
git commit -m "atualizaçao"
git push origin main
